# 肽毒性生物学背景

## 毒肽的一般特征

### 结构特征

| 特征 | 说明 | 典型例子 |
|-----|------|---------|
| **短链** | 通常 5-50 个氨基酸 | 蜂毒肽 (26aa) |
| **阳离子性** | 高比例 K、R 残基 | 抗菌肽 |
| **两亲性** | 亲水/疏水区域分离 | 穿孔肽 |
| **酰胺化 C 端** | C 端酰胺化增强活性 | 多种毒素肽 |

### 氨基酸组成特点

- **阳离子氨基酸** (K, R, H)：促进与细胞膜负电荷相互作用
- **疏水氨基酸** (A, V, I, L, F)：介导膜插入
- **芳香族氨基酸** (W, Y, F)：影响受体结合和毒性

## 毒性预测的生物学基础

### 作用机制

1. **膜作用机制**
   - 阳离子肽与细菌细胞膜负电荷相互作用
   - 疏水区域插入膜内部
   - 形成孔道导致细胞裂解

2. **MERCI Motif**
   - 基于已知毒素序列模式的匹配
   - 对已知毒素类型高特异性
   - 快速筛选大量序列

### 预测挑战

| 挑战 | 说明 |
|------|------|
| **假阳性风险** | 某些治疗性抗菌肽可能被误判为有毒 |
| **物种特异性** | 不同物种的毒素可能具有不同特征 |
| **浓度效应** | 低浓度可能无毒，高浓度有毒 |
| **新毒素** | 只能检测已知毒素 motif，新毒素可能漏检 |

## AAC/DPC 特征的生物学意义

### AAC (氨基酸组成)

反映序列的整体氨基酸分布：
- 阳离子氨基酸比例高 → 可能具有抗菌活性，但也可能有毒
- 疏水氨基酸比例高 → 影响细胞膜穿透能力
- 芳香族氨基酸 → 可能影响受体结合

### DPC (二肽组成)

反映相邻氨基酸的相互作用：
- 某些二肽 motif 是毒素的特征标记
- 反映序列的局部结构偏好
- 可捕捉氨基酸间的相互作用

## 在融合肽设计中的考虑

融合肽的毒性评估需要特别关注：

1. ** linker 区域**：可能引入新的毒性 motif
2. **各组分比例**：融合后各组分的暴露程度
3. **整体理化性质**：净电荷、疏水性变化
4. **浓度效应**：实际使用浓度下的毒性
